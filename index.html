Массив
Массив — это структура данных, используемая в программировании для хранения коллекции однотипных элементов. Массивы позволяют организовать данные в упорядоченную последовательность, что облегчает доступ и управление элементами
Упорядоченность:
Элементы массива хранятся в определенном порядке, и каждому элементу присваивается индекс, который указывает на его позицию. Индексы начинаются с 0 в большинстве языков программирования.
Однотипность:
В классических массивах все элементы имеют один и тот же тип данных (например, все элементы могут быть числами, строками и т. д.). В некоторых языках, таких как JavaScript, могут быть массивы с элементами разных типов.
Пример одномерного массива:
const fruits = ['apple', 'banana', 'cherry'];
Пример двумерного массива:
const matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ];
Основные операции с массивами:
  • Доступ к элементам: Элементы массива можно получать по индексу (например, fruits[0] вернет 'apple').
  • Изменение элементов: Можно изменять элементы массива, используя индексы (например, fruits[1] = 'orange').
  • Итерация: Можно проходить по массиву с помощью циклов (например, for, foreach и т.д.).
  • Добавление и удаление: Зависит от языка, но многие имеют встроенные методы для добавления и удаления элементов (например, push и pop в JavaScript).
Методы массива
Изменение массива
push(element): Добавляет один или несколько элементов в конец массива и возвращает новую длину массива.
  • const fruits = ['apple', 'banana'];
  • console.log(fruits); // ['apple', 'banana']
  • fruits.push('cherry');
  • console.log(fruits); // ['apple', 'banana', 'cherry']
pop(): Удаляет последний элемент массива и возвращает его
  • const fruits = ['apple', 'banana'];
  • console.log(fruits); // ['apple', 'banana', 'cherry']
  • const lastFruit = fruits.pop();
  • console.log(lastFruit); // 'cherry'
  • console.log(fruits); // ['apple', 'banana']
shift(): Удаляет первый элемент массива и возвращает его.
  • const fruits = ['apple', 'banana', 'cherry'];
  • console.log(fruits); // ['apple', 'banana', 'cherry']
  • const firstFruit = fruits.shift();
  • console.log(firstFruit); // 'apple'
  • console.log(fruits); // ['banana', 'cherry']
unshift(element): Добавляет один или несколько элементов в начало массива и возвращает новую длину массива.
  • const fruits = ['banana', 'cherry'];
  • console.log(fruits); // ['banana', 'cherry']
  • fruits.unshift('apple');
  • console.log(fruits); // ['apple', 'banana', 'cherry']
splice(start, deleteCount, item1, item2, ...): Удаляет, заменяет или добавляет элементы в массив, начиная с указанного индекса..
  • start - индекс с которого начнется изменение массива, "-1" - послений элемент массива, "-2" предпоследний и т.д.
  • deleteCount - количество удаляемых элементов включая start и элементы идущие за ним
  • item - новые элементы массива встают на место start
  • const fruits = ['apple', 'banana', 'cherry'];
  • console.log(fruits); // ['apple', 'banana', 'cherry']
  • fruits.splice(1, 1, 'orange');
  • console.log(fruits); // ['apple', 'orange', 'cherry']
slice(start, end): Возвращает новый массив, содержащий копию части оригинального массива.
  • start - индекс с которого начинается копирование
  • end - индекс на котором заканчивается копирование, не включая этот элемент (не обязательный)
  • const fruits = ['apple', 'banana', 'cherry'];
  • const citrus = fruits.slice(1);
  • console.log(citrus); // ['banana', 'cherry']
